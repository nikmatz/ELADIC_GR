# ELADIC_GR — Ψηφιακό Συνοδευτικό Υλικό

Ψηφιακές δραστηριότητες που συνοδεύουν το βιβλίο **«Μαθηματικά Ι»** του Ν. Ματζάκου.

### Περιεχόμενα

| Φάκελος | Περιγραφή |
|---------|-----------|
| `quizzes/` | Κουίζ αυτοαξιολόγησης (21 κεφάλαια, 10+ ερωτήσεις) |
| `animations/` | Διαδραστικές animations (CAS & AI) |
| `_shared/` | Κοινά στυλ (book.css) |

### GitHub Pages

Διαθέσιμο στο: **https://nikmatz.github.io/ELADIC_GR/**

### Δομή Κουίζ

```
quizzes/
  ch00/index.html   Μαθηματικό Υπόβαθρο
  ch01/index.html   Γραμμικοί Χώροι
  ch02/index.html   Άλγεβρα Πινάκων
  ...
  ch19/index.html   Διαφορικές Εξισώσεις
```

Κάθε κουίζ περιλαμβάνει 10–12 ερωτήσεις (πολλαπλής επιλογής & σωστό/λάθος)
με άμεση ανατροφοδότηση και επεξήγηση για κάθε ερώτηση.
